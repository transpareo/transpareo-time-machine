# Code of Conduct

This project adopts version 2.1 of the Contributor
Covenant. The full text lives at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>;
the short version is:

- Be respectful of everyone who participates,
  regardless of background, identity, or experience
  level.
- Disagreement on technical decisions is fine; making
  someone feel unwelcome over them is not.
- Maintainers will act on concerns brought to them in
  good faith.

## Scope

Applies to all project spaces: this repository (issues,
pull requests, discussions, code review), any chat
channel run by the maintainers, and project-related
behaviour in public spaces (conferences, social media)
when someone is representing the project.

## Reporting

Concerns about a participant's conduct can go to
**conduct@transpareo.com**. Reports are read by the
maintainers and treated as confidential. We acknowledge
within 3 business days.

Security vulnerabilities go to a different inbox; see
[SECURITY.md](SECURITY.md).

## Enforcement

Maintainers apply the four-step enforcement ladder from
the Contributor Covenant (correction, then warning,
temporary ban, permanent ban). Decisions are not made
unilaterally by a single maintainer when a permanent
ban is on the table.
